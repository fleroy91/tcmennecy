== TC Mennecy Refinery CMS

1. Modify files in the github repo
2. Commit to master automatically deploys to Heroku
3. Update the CMS data via the CMS Admin console
  - https://tc-mennecy-fr.herokuapp.com/refinery
  - Login : `admin`
  - Password : `admin`
4. When done, use SiteSucker to download all the pages and images to your local hard drive
5. Push the content with FileZilla to the FTP website of the FFT
